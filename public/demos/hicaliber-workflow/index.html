Overview

From messy listing operations to structured workflow systems.

This demo turns a messy real estate listing operation into a structured workflow system: relational data, lifecycle states, automation rules, platform mapping, scenario simulation, and handover-ready documentation.

Built as a systems design exercise for a CRM & Workflow Automation Specialist role focused on Airtable and Monday.com.

How to use this demo

Use the sidebar to navigate each layer of the system design. Start with Data Model to see the relational structure, then follow the flow through State Machine → Rules Engine → Platform Mapping → Simulation → Handover Notes → Dashboard.
