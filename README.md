# Esther AI Builder Site

Personal Brand Hub + AI Builder Lab for Esther.

Core line:

> I just turn the problems I run into into tools.

## Stack

- Next.js
- TypeScript
- Tailwind CSS

## v0.1 Goal

Build a static personal builder site with:

- homepage sections
- LIVE / DEMO / NEXT Lab groups
- static demo imports under `public/demos`
- responsive layout
- SEO metadata
- no `file://` links
- no fake metrics or fake subscriber counts

## Local Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
```
